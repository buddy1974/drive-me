# User Flow — Drive Me

## Overview
The user journey from onboarding to completed ride.

---

## 1. Onboarding
- [ ] Splash screen
- [ ] Enter phone number (CMR format validation)
- [ ] Receive OTP via SMS
- [ ] Verify OTP
- [ ] Enter name (optional)
- [ ] Home screen

## 2. Booking a Ride
- [ ] Enter pickup location (map pin or search)
- [ ] Enter drop location (map pin or search)
- [ ] See estimated price
- [ ] Confirm booking
- [ ] Waiting for agent acceptance

## 3. Ride in Progress
- [ ] Real-time agent location on map
- [ ] Agent en route → Arrived → In progress
- [ ] ETA display
- [ ] Cancel option (before acceptance)

## 4. Payment
- [ ] Ride completed
- [ ] Payment screen (MTN MoMo or Orange Money)
- [ ] Enter MOMO PIN on phone (USSD push)
- [ ] Payment confirmed

## 5. Post-Ride
- [ ] Rate the agent (1–5 stars)
- [ ] Ride summary / receipt
- [ ] Return to home
